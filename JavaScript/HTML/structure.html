  • HTML pages are text documents.
  • HTMLS uses tags, which are chacters that sit inside angled brackets. They act like containers and tell you something about the information that lies between them.
  • Tags are often reffered to as elements
  • Tags usually come in pairs. The openeing tag denotes the start of a piece of content; the colosing tag denotes the end
  • Opening tags can carry attributes, which tell us more about the content of that element.
  • Attributes require a name and value.
  • To learn HTML you need to know what tags are available for you to use, what they do, and where they can go.
