Undefined Variables

When you create a variable and don't assign any value to it, it will be of undefined type.
The reason we use undefined is that when you create a vfunction that relies on some variable, and it is undefined, that means nothing happened to that variable yet.

Regular Expressions

JS includes what is known as Regular Expressions,
which are used to search for specific patterns of characters or a sequence of characters.
There are two ways to create a regular expressions:

1. You can create what is known as the literal string version
var str = "This is a sample string";
var re = /Sample/i;

2. Create the regex object
var str = "This is a sample string";
var re2 = new RegExp("Sample", "i");
